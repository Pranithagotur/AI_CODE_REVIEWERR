#  AI Code Reviewer

An AI-powered code review platform that analyzes source code, detects bugs, identifies security vulnerabilities, highlights coding best practices, and provides intelligent suggestions using Google's Gemini AI.

## 🌐 Live Demo

**Frontend:** https://ai-code-reviewerr.vercel.app

**Backend API:** https://ai-code-reviewerr-umxn.onrender.com

##  Features

*  AI-powered code review using Gemini AI
*  Security vulnerability detection
*  Code quality analysis
*  Best practice recommendations
*  Rule-based validation engine
*  Analytics Dashboard
*  Knowledge Base
*  Review History Tracking
*  Modern responsive UI
*  Cloud deployment using Vercel & Render

##  Tech Stack

### Frontend

* React

### Backend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Pranithagotur/AI_CODE_REVIEWERR.git
cd AI_CODE_REVIEWERR
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Runs on:

```bash
http://localhost:5173
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

Runs on:

```bash
http://localhost:5000
```

---

## Environment Variables

### Backend (.env)

```env
PORT=5000
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_api_key
```

### Frontend (.env)

```env
VITE_API_URL=https://ai-code-reviewerr-umxn.onrender.com
```
